# FiveM-Banking
FiveM Economy/Banking storing data via SteamID in JSON format. This was meant to be a concept that turned into chicken scratch code release.

## Features
* Pay Players
* Admin Pay Players
* Server Balance
* Income Tax
* Sales Tax
* Horrible Lua
* Unemployement (Automatic Payments)
* Cash
* Bank Commands

## Commands
* /pay ID Amount
* /adminpay ID Amount
* /serverbalance
* /balance or /balance id
* /deposit amount
* /withdraw amount
